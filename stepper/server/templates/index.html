• calibrationPlatform.local/go2pose_mm/<pose_mm>
    • Goes to a certain pose, regardless of the current pose. Takes the wanted pose in mm as input
  • calibrationPlatform.local/calibrate
    • Calibrates the target bed. Takes no input.
  • calibrationPlatform.local/getCurrentHeight_mm
    • Returns the current pose in mm. Takes no input
  • calibrationPlatform.local/moveStepper_mm/<up_or_down>/<mm>/
    • Moves the stepper from the current pose. Takes direction and mm travel as input.
